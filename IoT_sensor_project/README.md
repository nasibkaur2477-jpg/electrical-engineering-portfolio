# Embedded IoT Environmental Monitoring System

This project demonstrates the design of an embedded hardware platform used to monitor environmental conditions.

## Hardware

- ESP32 microcontroller
- Temperature and humidity sensors
- Analog signal conditioning circuits

## Engineering Work

- Designed schematic using KiCAD
- Created PCB layout
- Simulated power circuits using Ngspice

## Applications

- Smart agriculture
- Environmental monitoring
- IoT sensor networks
